# Command Word Coach v0.1.23

Archived before the v0.1.24 CSS split refactor.

v0.1.23 separated configuration/runtime maintenance concerns while keeping the main UI and behaviour stable:

- `app-config.js` for version, URL constants, and neutral command-groove defaults;
- `app-maintenance.js` for runtime version display and neutral groove safety;
- neutral helper-level command-word defaults;
- `CODEMAP.md` documenting feature ownership and future target modules.

The full v0.1.23 source is recoverable from Git history immediately before the v0.1.24 release.
