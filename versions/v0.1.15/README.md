# Command Word Coach v0.1.15

Archived before the v0.1.16 command-groove fallback fix.

v0.1.15 kept command-word groove guidance in the top-right coach panel, but the HTML and JavaScript fallback still used `Reason → Effect → Link` as a generic default. v0.1.16 changes that fallback to neutral command-word guidance and keeps `Reason → Effect → Link` only for `why` questions.

The full v0.1.15 source is recoverable from Git history immediately before the v0.1.16 release.
